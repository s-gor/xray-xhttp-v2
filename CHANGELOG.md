# Changelog

Все заметные изменения проекта фиксируются в этом файле.

Формат основан на принципах Keep a Changelog. Версии проекта оформляются по схеме Semantic Versioning.

## [Unreleased]

## [2.1.0] - 2026-06-24

### Added

- Добавлен автоматический установщик Xray + XHTTP + Nginx + Let's Encrypt.
- Добавлен скрипт полного удаления с резервным копированием.
- Добавлены отдельные инструкции One-line Install и One-line Uninstall.
- Добавлена обработка лимита Let's Encrypt `rateLimited` с выводом времени `retry after`.
- Добавлена автоматическая проверка и установка cron-задания `acme.sh`.
- Добавлена финальная проверка служб, портов, сертификатов и клиентской ссылки.
- Добавлен раздел диагностики типичных ошибок.

### Changed

- Обновлён синтаксис HTTP/2 в Nginx:
  - удалено `listen 443 ssl http2;`;
  - используется `listen 443 ssl;` и `http2 on;`.
- Исправлена последовательность установки Xray, создания рабочих значений и конфигурации Nginx.
- Рабочие значения теперь сохраняются в `/root/xray-xhttp-values.env`.
- Xray слушает только локальный адрес `127.0.0.1:8443`.
- Параметр XHTTP `mode` не задаётся на сервере и может тестироваться на стороне клиента.
- Обновлены README и Quickstart, команды и проверки приведены к единому виду.
- Перед большими конфигурационными блоками добавлена инструкция копировать блок целиком.

### Fixed

- Исправлен ввод домена при запуске установщика одной командой.
- Исправлено завершение текущего SSH-сеанса при ошибке внутри скрипта.
- Исправлена ситуация, когда Xray не устанавливался, но инструкция переходила к созданию конфигурации.
- Исправлено создание пустых переменных `XHTTP_LOCAL_PORT`, `XHTTP_PATH` и `USER_UUID`.
- Исправлена ошибка Nginx `invalid port in upstream "127.0.0.1:"`.
- Исправлена ошибка HTTPS `502`, возникавшая при неработающем Xray.
- Исправлена ложная остановка после успешной установки TLS-сертификата.
- Исправлена проверка cron-задания `acme.sh`.
- Исправлены проверки с `grep -q` при включённом `pipefail`.
- Исправлен вывод финальной проверки: каждая проверка теперь показывает `OK` или `FAIL`.
- Исправлено предупреждение Nginx об устаревшей директиве `listen ... http2`.
- Исправлена диагностика ошибок: теперь выводятся команда, номер строки, код возврата и путь к журналу.

### Documentation

- Полностью переписан `README.md`.
- Полностью переписан `STEP-02-QUICKSTART.md`.
- Добавлены инструкции запуска установщика и удаления из обычной и root-сессии.
- Добавлены пояснения для ошибок:
  - `xray: command not found`;
  - `invalid port in upstream`;
  - HTTP `502`;
  - `rateLimited`;
  - `listen ... http2 is deprecated`.
